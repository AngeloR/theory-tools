# MVP Scope

## Supported Scales (Initial)

- Major
- Natural Minor
- Harmonic Minor
- Melodic Minor (Jazz melodic minor only)

Scales must support **variable length** for future expansion
(e.g. blues, pentatonics).

## Instrument

- Guitar only
- 6 strings
- Standard tuning (E A D G B E)
- 24 frets
- No capo
- No left-handed mode

## User Inputs

- Root note dropdown (explicit spelling)
- Scale dropdown

## Rendering

- Entire neck displayed (frets 1â€“24)
- High E string at top
- Horizontal scroll on mobile
- Only in-scale notes rendered

## Explicit Non-Goals (MVP)

- No audio
- No modes
- No arpeggios
- No chord progressions
- No persistence or sharing
- No accessibility accommodations
